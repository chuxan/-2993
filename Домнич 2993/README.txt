Для того что бы запустить проект на локальной машине откройте папку с проектом - /html - /index.html - двойной клик.
Либо откройте проект в Visual Studio Code. Так же выберите /html - /index.html, Затем F5, выбираете браузер.
При наличии плагина Live Server проект так же можно запустить через него.
